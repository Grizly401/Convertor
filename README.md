# HTML to PDF Converter

Простое приложение для конвертации HTML в PDF с двумя режимами работы.

## Возможности

- Конвертация локальных HTML файлов
- Конвертация веб-страниц по URL
- Два режима конвертации: быстрый и продвинутый
- Автоматическое определение типа контента
- Высокое качество PDF (300 DPI)

## Режимы работы

### Быстрый режим (QTextDocument)
- Мгновенная конвертация
- Базовая поддержка HTML
- Минимальное использование ресурсов
- Идеален для простых документов

### Продвинутый режим (QWebEngine)
- Полная поддержка CSS3 и JavaScript
- Загрузка внешних ресурсов
- Движок Chromium
- Подходит для сложных веб-страниц

## Как использовать

1. Выберите режим конвертации
2. Введите путь к файлу или URL в поле ввода
3. Нажмите "Конвертировать в PDF"
4. Выберите место сохранения

## Сборка

Требования:
- Qt 6.9.3+
- CMake 3.16+
- Компилятор C++17

## Планируемые обновления

- Многопоточная конвертация (без блокировки интерфейса)
- Пакетная обработка файлов
- Настройка качества PDF
- Темная тема интерфейса

## Технические детали

Приложение использует:
- Qt6 для графического интерфейса
- QTextDocument для быстрой конвертации
- QWebEngine для продвинутой конвертации
- CMake для системы сборки
